5.2 Часто задаваемые вопросы

Вопрос	Ответ
Как обновить программу?	sudo apt update && sudo apt upgrade -y
Где найти логи?	/var/log/[название_программы]/
Как изменить порт?	В конфигурационном файле /etc/[название]/config
